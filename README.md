# cn-site-archive

一个用于归档和发布多个独立 HTML 页面的通用仓库。本仓库不针对任何特定域名或网站，旨在提供一种简洁、结构化的方式保存和共享独立的网页内容。

## 项目简介

本项目以静态 HTML 页面归档为主要目的。页面内容可能涵盖各类信息、文档或工具页面，所有页面均为独立文件，不依赖后端服务或数据库。仓库本身不承担内容审核、推广或运营角色，仅作为存放与分发 HTML 资料的场所。

## 目录结构

```
.
├── README.md           # 本文件
├── pages/              # 存放所有归档 HTML 页面
│   ├── example.html    # 示例页面
│   └── ...
└── assets/             # 可选的共享资源（如 CSS、图片、脚本等）
    └── ...
```

- `pages/`：所有独立 HTML 页面均放置在此目录下，每个文件对应一个页面主题。
- `assets/`：如果多个页面共享相同样式或资源，可统一存放于此，避免重复。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，推荐将样式和脚本内联，以确保离线查看时依旧可用。
- 页面命名应简洁明了，能反映内容主题，但不得包含具体域名或商业标识。
- 归档页面不代表任何官方立场，内容来源与准确性由提交者自行负责。

## 维护说明

- 本仓库仅接受独立 HTML 页面的提交，不包含动态内容、重定向或外部依赖。
- 提交新页面时，请确保文件格式正确、无恶意代码，并放置在 `pages/` 目录下。
- 仓库维护者保留对内容进行筛选、修改或删除的权利，但不会主动推广或推荐任何页面。
- 如发现页面存在技术问题（如加载失败、链接错误），欢迎通过 Issue 反馈。

## 贡献

欢迎提交新的归档页面。请遵循以下简单规则：

1. 页面必须为纯 HTML，不包含服务器端脚本。
2. 文件命名使用小写字母、数字和连字符，后缀为 `.html`。
3. 提交前请确保页面在本地浏览器中可正常打开。
